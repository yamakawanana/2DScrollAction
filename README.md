【横スクロールゲーム】

このゲームは、敵を避けるか踏んで倒しながら右方向へ進み、ゴールの旗に触れるとステージクリアとなります。
ステージは全部で3ステージあります。
プレイヤーはダメージを受けてHPが0になるか画面下に落下すると死亡し、3回死亡するとゲームオーバーになります。

【操作方法】

右移動：→キー
左移動：←キー
ジャンプ：Ｚキー または ↑キー
ダッシュ：Ｘキー
一時停止：Escキー
